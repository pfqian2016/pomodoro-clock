### 番茄钟
根据自己的习惯来调节工作与休息时间，设置好后点击按钮开始计时，到达指定时间时，会自动切换为另外一种模式开始计时，如此循环往复。此仓库的代码没有经过babel转码，需在支持ES6的浏览器上运行才有效，访问可以前往[这里](http://www.qianpengfei.com/demo/pomodoro/)
